# デザイン定義

nofeed-twitter の UI に関する規約をまとめる。
※ そこまで厳格なものではなく、雰囲気で変えていい。

## 角丸

基本的に角は少し丸める。`rounded-lg` を使用すること。(雰囲気を見て変更して良い)

## 色

- 基本は slate 系列を使用すること。
- 注意・エラーは red 系列を使用すること。

### slate の濃さ

| 用途            | 値                             |
| --------------- | ------------------------------ |
| 背景            | `bg-slate-50`                  |
| 本文テキスト    | `text-slate-700`               |
| 見出し          | `text-slate-800`               |
| ラベル          | `text-slate-600`               |
| アイコン        | `stroke-slate-400`             |
| ボーダー        | `border-slate-400`             |
| ボーダー(focus) | `border-slate-500`             |
| リング(focus)   | `ring-slate-300`               |
| ボタン背景      | `bg-slate-700`（hover: `800`） |
| ボタン文字      | `text-slate-100`               |

### red の濃さ

| 用途            | 値               |
| --------------- | ---------------- |
| エラーテキスト  | `text-red-500`   |
| ボーダー        | `border-red-400` |
| ボーダー(focus) | `border-red-500` |
| リング(focus)   | `ring-red-300`   |
